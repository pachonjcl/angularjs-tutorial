Introduccion

AngularJS es: Un framework estructural para aplicaciones web dinamicas (MVC). Permite crear aplicaciones de una sola pagina. Permite utilizar elementos HTML y tus propios componentes. Permite una facil conexion entre el controlador y la vista. Utiliza la inyeccion de dependencias. Es amigable al momento de hacer las pruebas. Es bueno para el prototipado rapido. Es utilizado en mas de medio millon de paginas como Youtube y Upwork. Tiene un mayor mercado que otros frameworks para desarrollo web.
